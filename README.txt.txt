Since this projected is implemented on Google Colab, there is no additional default value or terminal command that need to be entered. One can simply run this .ipynb file on their Jupyter Notebook or Google Colab. 

The user MUST modify the PATH at the very top of the code, according to where the "train-images-idx3-ubyte.gz" folder is located on the google drive. The important thing is, in my code, every code path variable is different 
and adressing to different .zip files. Therefore, one should extract the main data folder into 4 subzip folders.
